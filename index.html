About Me

Halo, saya Rifqi Ardian. Saya belajar di SMKN 1 Kota Bengkulu, jurusan Pengembangan Perangkat Lunak dan Gim (PPLG). Saya fokus pada pengembangan aplikasi web dan mobile, serta senang membangun produk yang sederhana, cepat, dan bermanfaat. Perhatian saya terutama pada clean code, pengalaman pengguna, dan pemecahan masalah yang nyata.

Teknologi yang sering saya gunakan meliputi Laravel, Tailwind CSS, dan MySQL untuk web, serta Java untuk pengembangan aplikasi Android. Saya juga aktif mengelola kode di GitHub dan terus belajar hal-hal baru agar kualitas proyek semakin baik. Silakan cek bagian Projects untuk melihat karya saya, atau menuju Contact jika ingin bekerja sama.
